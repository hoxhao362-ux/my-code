# 期刊平台后端需求（优化版摘要）

## 文档目的与范围
- 目的：将前端文档中与后端实现相关的要点提炼为一致、可执行的规范，便于后端快速落地。
- 范围：角色与权限边界、枚举字段统一规范、返回结构示例、兼容与影响点说明。
- 统一约定：
  - 所有与状态/阶段相关的枚举值统一为全小写英文 snake_case。
  - 相同语义在所有接口中保持一致（status、reviewStage、record.stage）。
  - 下文提供标准枚举与返回示例；文末保留完整接口清单作为附录。

## 角色与权限边界
- Editor（正式编辑）：终审判定、全流程统筹、团队权限分配；拥有最高权限，负责稿件全流程最终决策。
- Associate Editor（副编辑）：稿件初筛、推荐审稿人、审核评审意见；无最终决策权，仅提供专业领域参考建议。
- EA & AE（Editorial Assistant & Advisory Editor，编辑助理/顾问编辑）：流程文书处理、进度跟进、作者基础沟通；学术质量把控与争议稿件专业咨询；不参与评审与决策。
- 任命与准入：所有角色均不可自由注册，必须走“申请/提名 → 审核 → 任命 → 激活”流程。
- 编委会管理：编委会统一管理编辑角色、权限与流程；编委会成员 = Editor；一个编委账号 = 一个 Editor 角色；非 Editor 不可进入编委会模块。

## 枚举字段统一规范

### 稿件状态 status（后端统一使用英文）
| 中文 | 英文值 | 说明 |
|------|--------|------|
| 待提交 | pending | 草稿未提交 |
| 已提交 | submitted | 作者已提交 |
| 审稿中 | under_review | 正在审稿 |
| 已审核 | review_completed | 评审流程完成 |
| 需修改 | revision_required | 返回作者修订 |
| 已通过 | accepted | 通过 |
| 未通过 | rejected | 拒稿 |
| 已发表 | published | 已出版/发表 |

说明：后端需统一返回以上英文值，替代中文或中英文混用。

### 审稿阶段 reviewStage
| 中文 | 英文值 |
|------|--------|
| 初审 | initial_review |
| 复审 | peer_review |
| 终审 | final_decision |

说明：record.stage 与 reviewStage 使用同一套取值，遗留的 “Re-review” 统一为 peer_review。

### 返回数据示例
```json
{
  "id": 1,
  "title": "稿件标题",
  "status": "accepted",
  "reviewStage": "peer_review",
  "reviewResult": "accepted",
  "reviewHistory": [
    {
      "stage": "initial_review",
      "status": "passed",
      "comment": "审核意见"
    }
  ]
}
```

## 三十、稿件状态流转说明

```
投稿提交 → pending_initial_review (待初审)
    ↓
初审阶段:
├── initial_review_passed (初审通过)
├── initial_review_revision (初审退回修改)
└── initial_review_rejected (初审拒稿)
    ↓
同行评审阶段:
├── pending_peer_review (待同行评审)
├── under_peer_review (同行评审中)
├── review_completed (评审完成)
└── revision_required (需要修稿)
    ↓
终审阶段:
├── pending_final_decision (待终审)
├── under_final_decision (终审中)
├── final_decision_accepted (终审通过)
├── final_decision_revision (终审修改后录用)
└── final_decision_rejected (终审拒稿)
    ↓
出版阶段:
├── pending_acceptance_confirmation (待录用确认)
├── pending_copyright (待版权协议签署)
├── pending_proof (待校样确认)
├── pending_publication (待出版)
└── published (已出版)
```
