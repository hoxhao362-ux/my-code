# 问题：
1. 注册时是否需要填写邮箱和验证码？
2. 个人信息页面修改邮箱和手机号时需要前端先正则匹配是否符合格式
3. 注册时管理员辨识密码字段改一下，改成邀请码，不让外部识别。然后这个邀请码不填就是“普通用户”，填写管理员邀请码就是“管理员”，审核员邀请码就是“审核员”
4. 可能需要管理员后台，审核员后台，作者后台，网站主站（就是所有人都可以使用的）
5. 所有账号均可登录网站主站（前提是在网站主站的登录界面登录），然后管理员后台，审核员后台，作者后台都需要登录各自的后台界面。
6. 前端请先实现密码的加密传输。


# 注意事项
1. 有什么功能需要请求后台功能实现的，写在 后端需求.md 中，写好之后前端只需要调用对应的 API 即可。
2. 后端API地址为 http://localhost:8000 ，需要查看接口文档，最好使用开发环境的接口文档（启动后端服务器后网址http://localhost:8000/api/docs）
3. 目前后端python版本3.12，需要第三方库，位于 backend/requirements.txt 中
    使用方法：
    ```    
    # 打开cmd，进入backend目录
    conda activate <your_env_name>  # 激活conda环境
    pip install -r requirements.txt
    ```

4. 启动后端方法：
    ```
    # 打开cmd，进入backend目录
    conda activate <your_env_name>  # 激活conda环境
    uvicorn main:app --reload
    ```
